# ESTUDIOS DE TRADING — Institutional Trading OS

Plataforma web premium para presentar un plan de estudio de trading institucional, roadmap de aprendizaje, backtesting, journal y un Asistente de Análisis informativo.

## Seguridad del asistente

El Asistente de Análisis no ejecuta operaciones, no abre compras, no abre ventas, no cierra operaciones y no se conecta a dinero real. Solo genera análisis educativo, informativo y reportes simulados.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui-style components
- Framer Motion
- lucide-react
- Recharts

## Ejecutar el proyecto

Instala dependencias:

```bash
npm install
```

Levanta el entorno local:

```bash
npm run dev
```

Abre:

```bash
http://127.0.0.1:3000
```

Validar producción:

```bash
npm run build
```

Revisar código:

```bash
npm run lint
```
